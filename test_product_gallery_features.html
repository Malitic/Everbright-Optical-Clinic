üîó API Endpoints

Product Management APIs:

  • GET /api/products - List all products (with filters)
  • POST /api/products - Create new product
  • GET /api/products/{id} - Get single product
  • POST /api/products/{id} - Update product (with _method=PUT)
  • DELETE /api/products/{id} - Delete product
  • POST /api/products/{product}/toggle-status - Toggle active status
  • GET /api/product-categories - Get product categories

Branch Stock APIs:

  • GET /api/branch-stock - Get branch stock data
  • POST /api/branch-stock - Update branch stock
