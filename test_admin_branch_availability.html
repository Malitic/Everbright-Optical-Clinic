How to Access This Dashboard:

  1. 1. Start the backend server: cd backend && php artisan serve
  2. 2. Start the frontend: cd frontend-- && npm run dev
  3. 3. Login as an Admin user
  4. 4. Navigate to Admin → Branch Availability
  5. 5. You'll see this comprehensive view of all products across all 4 branches

Dashboard Features:

  • • Real-time Stock Monitoring: See current stock levels across all branches
  • • Reservation Tracking: Monitor reserved quantities per branch
  • • Low Stock Alerts: Identify products that need restocking
  • • Branch Comparison: Compare availability across all 4 branches
  • • Search & Filter: Find specific products or filter by stock status
  • • Summary Statistics: Get overview of total inventory across all branches
