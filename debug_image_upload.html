Image Upload Debug Tool

Backend Status Check

Backend URL: http://127.0.0.1:8000

API Endpoint: POST /api/upload/image

Frontend URL: http://localhost:8081

Click here or drag and drop an image file

Max size: 2MB | Supported: JPEG, PNG, JPG, GIF

Debug Steps:

  1. Make sure backend is running: cd backend && php artisan serve
  2. Make sure frontend is running: cd frontend-- && npm run dev
  3. Check browser console for errors (F12)
  4. Check network tab for failed requests
  5. Verify you're logged in as admin
  6. Check if CORS is properly configured

Common Issues:

  • CORS Error: Backend not allowing frontend requests
  • Authentication Error: Not logged in or token expired
  • File Size Error: Image too large (>2MB)
  • File Type Error: Not a valid image format
  • Storage Error: Uploads directory not writable
