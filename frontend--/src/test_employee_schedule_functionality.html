üìä API Endpoints

GET /api/staff-schedules/all
- Fetches all schedules across all branches
- Admin-only access
- Returns schedules with relationships

GET /api/staff-schedules/branch/{branchId}
- Fetches schedules for specific branch
- Returns grouped schedule data

POST /api/staff-schedules
- Creates or updates schedules
- Admin-only access

DELETE /api/staff-schedules/{scheduleId}
- Deletes schedule
- Admin-only access

GET /api/staff-schedules/change-requests
- Fetches change requests
- Supports filtering by status, role, branch

PUT /api/staff-schedules/change-requests/{id}/approve
- Approves change request
- Admin-only access

PUT /api/staff-schedules/change-requests/{id}/reject
- Rejects change request
- Admin-only access
