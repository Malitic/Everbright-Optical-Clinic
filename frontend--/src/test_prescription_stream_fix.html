Previous Error: "TypeError: Failed to execute 'text' on 'Response': body stream already read"
Root Cause: The code was trying to read the response body multiple times (first as JSON, then as text), which is not allowed in the Fetch API.
Fix Applied: Used `response.clone()` to create a copy of the response, allowing us to read it multiple times safely.
